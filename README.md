# FeliCa IDm Reader

NFC enabled web application to read FeliCa student ID cards and extract IDm.

## Usage

Visit: https://NEDE-tikyu.github.io/felica-idm-reader/

**Requirements:**
- Android Chrome 93+
- NFC enabled device
- FeliCa card

## Files

- index.html - UI
- style.css - Styling  
- main.js - NFC reading logic
